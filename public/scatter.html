ScatterChart

The scatter chart is visualizing data with up to five dimensions. One dataset can contains five data entries (e.g. one data entry for each axis and one for the size as well as the shape of the dataset). So it is possible to show data records with up to five data entries and visualize the data in the three dimensional space through the position, size and shape of one entity.

An example of a dataset (with five data entries) can be viewed underneath: 

    var jsonData = [
        {
            "name":"Dataset",
            "values": [
                {"name":"X", "value":0.5},
                {"name":"Y", "value":0.7},
                {"name":"Z", "value":0.3},
                {"name":"size", "value":11},
                {"name":"shape", "value":"sphere"}
            ]
        }
    ]

Each value in the "values" array has to end with the specific axis name or the value name "size" and "shape. So if you plan on giving your axis a certain name, dont forget to append the name of the axis (e.g. "moneyX" or "timeaxisY" or "ageZ"). The fourth value should also end with the string "size", in order to also give the opportunity of setting indivdiual naming conventions. (e.g. "marketShareSize) The fith data entry is optional. If no "shape" is set in the dataset, the "sphere" shape will be taken automatically. It is possible to set one of the following three shapes:

  • cone
  • diamond
  • sphere
