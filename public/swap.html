Data Swapping

How to swap the data of a chart during runtime

If you want to change the data a chart is based on, you can easily create a function that does that and trigger that function by using a button for example. First you need to create said function 

var swap = function () {
    chart.data(newdata);
    chart.swapData();
};

The make use of that function, the chart need to have a variable assigned to it. The chart has to be created like shown under usage but given a name var chart = anchart3d.createChart() [...]

The chart is then accessed via the swapfunction and - in this case - the data method is called again in order to change the data source. The last step would be - as mentioned before - to trigger that function using a button or link or via callback <a href="javascript:;" onClick="swapPie()">swap data</a>
