Data Swapping

How to swap the data of a chart during runtime

To make use of the swapping data feature, the chart has to first be initialized under a variable. You can find a guideline in the section usage on how to initialize and generate a chart. In this example we will initialize the chart under the variable "myChart", in order to later swap the data on the fly: var myChart = anchart3d.createChart() [...]

If you now want to change the data of a chart, you can use the swapData() method of the fluent api. Before you can apply the method, you have to first define the new data, which should be exchanged with the "old" data. Therefore, you simply use the data() method again, and pass the JSON - which contains your new data - as an argument. Finally, you can use swapData() to visualize the "new" data during runtime. 

var swapOldData = function () {
    myChart.data(myNewData);
    myChart.swapData();
};

The function swapOldData, which can be seen in the second code snippet, can now be used in different ways. One example could be the usage of a button: <button onclick="swapOldData()">Swap data</button> As you can see, we have added an "onClick" event to the button, which will call our function from the first figure!
