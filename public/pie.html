PieChart

The pie chart is visualizing data with two dimensions. One dataset can contain at most two data entries (e.g. one data entry for the radius of the segment and one data entry for the height of the segment). So it is possible to show data records with two data entries and visualize the difference of the data entries with the radius and height of each segment.

An example of a dataset (with two data entries) can be viewed underneath: 

    var jsonData = [
        {
            "name":"Austria",
            "values": [
                {"name":"Population", "value":8747000},
                {"name":"Average salary", "value":2670}
            ]
        }
    ]

As it is shown here, the dataset can contain two data entries, which are then displayed within the radius and height of a segment.
