anchart3d

...is an open source library for interactive three dimensional charts in the browser

This library has been created as part of a project on the FH Joanneum Graz in the degree program Information management by the following students:

  • Amar Bajric
  • Michael Fuchs
  • Timo Hasenbichler
The project was supervised by Dipl.-Ing. Dr.techn. Peter Salhofer.

The library is one of just a few contender in the world of visualizing data in three dimensions using not only perspectives but also interactions. Also it is completely free, since it is open source. Our goal was to provide a good base for visualizing data in three dimensional space in the browser and to show what the library is capable of. Moreover, it should enable the community to contribute to that, and make anchart3d a worthy contender in the world of data visualization.

The library is build using ES6 and relies on the three.js library to render the charts in the browser.
