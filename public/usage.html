Usage

How to get started using anchart3d

These are the basic steps you need to go through - if you want more control please see the configuration.

Including the script in the html file

Setting up anchart3d is really simple and straightforward. First, you need to download the anchart3d.js and place it in your project folder - for example in a subfolder called "js". After that you simply need to include it in your html like this: <script type="text/javascript src="js/anchart3d.js"></script>

You do not need to include anything else - all dependecies are already bundled into that one file and you're ready to go. Have a look at the configuration to find out more.

Then you need to include a <div> somewhere on your page. The div has to have an id with a name of your liking. For example, you could name it simply "chart": <div id="chart"></div>

Generating a chart via the fluent api

The last step would be to create a <script> tag as last element within the body (just before </body>) and call the actuall chart. 

anchart3d.createChart("chart")
  .setConfig(configuration)
  .pieChart()
  .data(jsonData)
  .draw();

As you can see, calling the chart takes a couple of additional informations. First you call the createChart() method from the public anchart3d variable, and pass the id of the div, which we have created earlier, as an argument. Then you pass an array with your configuration to setConfig(). This is optional and described here. The third method defines what kind of chart should be created. In this example we want to create a pie chart, so we call pieChart(). Another very important "ingredient" to visualize data, is the actual data you want to visualize. This specific data has to be stored in a JSON and then passed to the data() method. The data() method takes an optional second parameter, which is of type string, that you can use to sort the data. The string has to be equal with the name of one the data entries in the dataset(s). In order to look up on how you have to structure the JSON for your data, visit the specific link of the chart you want to use. There you can find a description on how to define the data in the JSON. The last method is called draw() (which has to be called last in the fluent api chain), which will initialize and generate the chart.
