usage

how to get started using anchart3d

These are the basic steps you need to go through - if you want more control please see the configuration.

Setting up anchart3d is really easy and straight forward. First, you need to download the anchart3d.js and place it your project folder - for example in a subfolder called "js". After that you simply need to include it in your html like this. <script type="text/javascript src="js/anchart3d.js"></script>

You do need to include anything else - all dependecies are already bundled into that one file and you're ready to go. Have a look at the configuration to find out more.

Then you need to include a <div> somewhere on your page. this div needs to have an id but how you name it is your choice. Let's say you name it "chart". <div id="chart"></div>

The last step would be to create a <script> tag as last element within the body (just before </body>) andcall the actuall chart. 

    anchart3d.createChart("chart")
      .setConfig(configuration)
      .pieChart()
      .data(jsonData)
      .draw();

As you can see, calling the chart takes a couple of additional information. First you call createChart() and pass it the id of div we created earlier. Then you pass an array with your configuration to setConfig(). This is optional and descrbed here. The configuration method takes an optional second parameter - a string that you can use to sort the data. That must be an category of your dataset - e.g. "income". The third method defines, what kind of chart is to be created. In this example we want to create a pie chart so we call pieChart(). Another very importent "ingredient" to visualize data is data which is to be in the form of a json and passed to data(). The last method is draw() (which must always be last) and that leads to your chart getting created.
