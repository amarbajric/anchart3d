Developing

How you can contribute to this project

Feel free to fork this project an participate in it's development.

You will need node.js and npm in order to run it.

Step 1:

> npm install

Step 2:

> npm start webpack watcher
> npm build quickly build the project
> npm minify minify using uglify
> npm test testing via jest
