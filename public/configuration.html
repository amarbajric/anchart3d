Configuration

How to configure anchart3d charts to your liking

Once you're done with the basic setup, as described here, you can start configuring using the fluent api based methods as described below.

You first need to create a so called configuration array that holds a set of configurations. var config = {param1:value1, param2:value2}

Of course this needs to placed inside the <script> tag we created as described under usage. Now let's have a look at the possible parameters and what changes they cause. The value is always a boolean.

details
When a segement gets clicked, a detail pane is displayed.
tooltip
When the curser hovers above a segment, a tooltip is shown.
legend
Displays a legend.
showOnScreenControls
Usually the chart can be interacted with using the mouse. Enabling this will display a element that allows moving the chart without using drag & drop functionality.
antialias
Smoother visuals but can cause problems on really old computers.
startAnimation
Wheter an animation should be used when the page is loaded. No animation means the chart is immediately displayed.
